# Chunk 07 — Exercise Types & Fasted Training

**SIT** — Sprint Interval Training: >100% VO2max, **8–30 sec** bursts + recovery.

**HIIT** — 80–100% VO2max, **60–240 sec** bursts + recovery.

**MICT** — Zone 2 steady state: **20–60 min**, 40–60% VO2max or **55–70% max HR**.

[17:54] **Fasted exercise:**
- Short training: fasted vs fed — little difference for fat oxidation.
- **~90 min moderate (MICT):** switchover — if fed, burn far less fat from 90 min onward vs fasted.
- **High intensity 20–60 min** (weights, sprints, kettlebells): switchover to fat burning comes **earlier** if fasted.

[19:53] If you ate before exercise → insulin up (especially carbs) → switchover delayed.

[20:14] **Sequence:** High intensity 20–60 min → Zone 2 cardio. **Fasted** = higher % body fat burned per unit time (if medically cleared).

[21:37] **Not for performance** — if goal is sport/competition, nutrition rules change completely.
