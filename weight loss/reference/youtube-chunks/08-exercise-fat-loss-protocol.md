# Chunk 08 — Tool: Exercise for Fat Loss

[21:20] **Simple protocol:** 3–4×/week — **high-intensity training** followed by **low-intensity (Zone 2)**, especially if fasted and physician-cleared.

[22:10] Why? **Neurons + adrenaline** — high effort deploys epinephrine; signals fat thermogenesis and oxidation.

[22:56] **Adrenaline is the final common path** for fat loss — whether shiver, sprint, or long bike ride.

**Protocol — Fat-Loss Accelerator:**
1. 20–60 min high intensity (weights, SIT, HIIT — shorter if harder)
2. Immediately → Zone 2 cardio (MICT)
3. Ideally fasted (fat loss goal only)
4. Repeat 3–4×/week

**App:** "Fat-Loss Accelerator" option in exercise logger.
