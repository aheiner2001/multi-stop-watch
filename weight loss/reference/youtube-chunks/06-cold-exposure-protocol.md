# Chunk 06 — Tool: Deliberate Cold Exposure (Shiver Protocol)

[13:42] To trigger shiver: **get in cold → get out (don't dry off) → get back in**. More shivering than staying in continuously.

[14:02] **Safety:** Very cold water can shock the heart / heart attack if not adapted. Start **uncomfortable but safe** (~60°F for many).

[14:35] **Frequency:** 1–3×/week (up to 5×) to accelerate fat loss.

[14:41] Get in until you **genuinely shiver** → out 1–3 min, **do not dry off** → back in → repeat shiver. **~3 repetitions** (3 in, 3 out).

[32:31] In/out pattern **avoids cold adaptation** so you continue to oxidize fat (vs adapting to constant cold).

**Protocol summary:**
| Parameter | Value |
|-----------|--------|
| Frequency | 1–5×/week |
| Temp | Uncomfortable; ~60°F start |
| Trigger | Genuine shiver |
| Out duration | 1–3 min, wet |
| Cycles | ~3 in/out |
