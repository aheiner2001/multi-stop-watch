# Utah Study Hours Tracker

Live site: https://aheiner2001.github.io/multi-stop-watch/

**GitHub Pages:** enable Pages from the branch that contains this `index.html` (usually `master` / `main`, root folder). The page injects a `<base href="…">` on `http`/`https` only so `styles.css` and scripts resolve correctly for project URLs like `/repo-name/` or `/repo-name` (no trailing slash). Local `file://` opens are unchanged.

Repository includes an empty `.nojekyll` so GitHub does not run Jekyll on the site.
