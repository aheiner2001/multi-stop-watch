# The Cut

Science-based 10-week cut tracker: **217 lbs → 160 lbs** at **1,250 cal/day**. Daily logging, food/macros, NEAT, and neuro protocols (Huberman Lab Essentials — Fat Loss framework).

**Live:** [https://aheiner2001.github.io/multi-stop-watch/weight%20loss/](https://aheiner2001.github.io/multi-stop-watch/weight%20loss/)

## Quick start

Open `index.html` locally, or use the live link above. No build step — vanilla HTML/CSS/JS with `localStorage`.

| Page | Purpose |
|------|---------|
| [index.html](index.html) | Home — today’s metrics & shortcuts |
| [daily.html](daily.html) | Daily Hub — log, food, NEAT, protocols, weight chart |
| [history.html](history.html) | Full timeline + CSV export |
| [maxes.html](maxes.html) | Weekly exercise PRs |
| [plan.html](plan.html) | Reverse-diet engine after goal |
| [guide.html](guide.html) | Science pillars & guardrails |

## Companion script

```bash
python3 cutting.py
```

Lean-mass decision tree aligned with dashboard guardrails (protein floor, lifting, fast days).
